# Wireframe: 生活场景 / 按问题学习

> 目标：让客户或小白从真实问题进入知识库，而不是先理解专业分类。

## 页面优先级

1. 这个场景适合谁
2. 先理解什么
3. 需要准备什么资料
4. 可以问顾问什么问题
5. 相关知识文章
6. 不能直接下结论的边界

## 列表页结构

```text
┌──────────────────────────────────────────────────────────────┐
│ Header                                                       │
├──────────────────────────────────────────────────────────────┤
│ 按问题学习                                                   │
│ 不知道该看哪个主题？先从你的生活场景开始。                    │
│ [搜索问题或关键词]                                            │
├──────────────────────────────────────────────────────────────┤
│ [我刚来加拿大]       [我想买第一套房]                         │

│ [快退休了]           [我是小企业主]                           │
└──────────────────────────────────────────────────────────────┘
```

## 详情页结构

```text
┌──────────────────────────────────────────────────────────────┐
│ Header                                                       │
├──────────────────────────────────────────────────────────────┤
│ 我想买第一套房，应该先准备什么？                              │
│ 适合：首次买房者、新移民家庭、正在比较首付和房贷的人。          │
│ Notice: 教育用途，不是房贷、税务或投资建议。                   │
├──────────────────────────────┬───────────────────────────────┤
│ 先理解这 5 件事              │ 需要准备的资料                 │
│ 1. 首付和现金流              │ - 收入证明                    │
│ 2. FHSA                      │ - CRA room                    │
│ 3. RRSP HBP                  │ - 银行 statement              │
│ 4. 房贷压力测试              │ - debt / credit info          │
│ 5. 保险和风险                │ - mortgage pre-approval info  │
├──────────────────────────────┴───────────────────────────────┤
│ 可以问顾问的问题                                              │
│ - 我是否符合 FHSA 基本条件？                                  │
│ - 我的首付资金在哪些账户？                                    │
│ - 房贷续约或利率变化会怎样影响现金流？                         │
├──────────────────────────────────────────────────────────────┤
│ 相关知识文章                                                   │
│ [FHSA] [首次买房] [注册账户总览] [房贷压力测试] [保险风险]      │
├──────────────────────────────────────────────────────────────┤
│ 不直接下结论                                                   │
│ 本页不会告诉你应该买哪套房、借多少、选哪家贷款或用哪个账户。     │
└──────────────────────────────────────────────────────────────┘
```

## 验收

- 用户不需要知道“注册账户 / 房贷 / 保险”这些分类，也能进入内容。
- 每个场景都要有资料准备和顾问问题。
- 每个场景都链接至少 3 篇知识文章。
- 页面不使用“客户教育”作为主标题，优先使用“生活场景”或“按问题学习”。
