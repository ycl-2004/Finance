# Wireframe: 首页 Dashboard

> 目标：让新用户 10 秒内知道这是中文加拿大金融学习中心，并立刻选择“开始学习”或“按问题进入”。

## 页面优先级

1. 开始学习
2. 生活场景 / 按问题学习
3. 主题知识库
4. 工具：术语表、资料清单
5. 案例练习
6. 教育边界

## 内容要求

- 首页不能是营销 landing page。
- 首页不能只是 8 个主题卡片。
- 首屏必须出现“开始学习”和“按问题学习”。
- 主题知识库是深度入口，不是唯一入口。
- 资料清单和术语表作为工具入口出现。

## 验收

- 用户不懂金融分类，也知道下一步点哪里。
- 客户可以从生活问题进入，而不是从专业术语进入。
- 页面气质专业、克制、可信，不像课程广告页。
