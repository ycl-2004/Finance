# Wireframe: 学习路线

> 目标：把 40 个 Markdown 文件变成小白可跟随的学习路径。

## 页面优先级

1. 当前适合谁学
2. 阶段顺序
3. 每阶段学习目标
4. 必读文章
5. 检查问题
6. 相关生活场景

## 阶段组件内容

每个阶段必须包含：

- 阶段编号。
- 阶段标题。
- 学习目标。
- 适合谁。
- 必读文章。
- 选读文章。
- 学完后应该能回答的问题。
- 相关生活场景。

## 验收

- 完全小白可以不看文件夹结构，直接开始。
- 每个阶段都有明确的“下一步”。
- 学习路线不是文章列表，而是学习任务列表。
