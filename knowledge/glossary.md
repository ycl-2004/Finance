# 金融术语表

## 规划和顾问

| Term | 中文 | 说明 |
| --- | --- | --- |
| Financial advisor | 金融顾问 | 泛指帮助客户处理金融产品或财务决策的人；实际权限取决于牌照、注册和所在省份规则。 |
| Financial planner | 财务规划师 | 侧重长期目标规划，例如退休、税务、保险、教育和传承。 |
| Wealth management | 财富管理 | 综合管理资产、投资、保险、税务、退休和传承。 |
| AUM | 管理资产规模 | Assets under management，通常指顾问或机构管理的客户资产规模。 |
| KYC | 了解你的客户 | 收集客户身份、目标、财务状况、投资经验、风险和时间周期。 |
| KYP | 了解你的产品 | 顾问或机构需要理解产品结构、费用、风险和适用场景。 |
| Suitability | 适合性 | 某个产品或策略是否适合特定客户。 |
| Risk tolerance | 风险承受能力 | 客户心理和财务上能承受投资波动或亏损的程度。 |
| Risk capacity | 风险承担能力 | 基于收入、资产、时间、责任和现金流的客观承受能力。 |
| Time horizon | 时间周期 | 资金距离使用还有多久。 |
| Liquidity | 流动性 | 资产能否快速、低成本地变成现金。 |

## 注册账户

| Term | 中文 | 说明 |
| --- | --- | --- |
| TFSA | 免税储蓄账户 | 供款不抵税，合规收益和提款一般免税。 |
| RRSP | 注册退休储蓄计划 | 供款通常可降低应税收入，提款通常纳税。 |
| FHSA | 首套房储蓄账户 | 合格首次买房者使用；合格供款可抵税，合格买房提款免税。 |
| RESP | 注册教育储蓄计划 | 为受益人未来高等教育储蓄，可关联政府教育补助。 |
| RDSP | 注册残障储蓄计划 | 为符合 Disability Tax Credit 资格人士长期储蓄，可关联政府补助和债券。 |
| RRIF | 注册退休收入基金 | RRSP 退休后常见转换形式，需按规则每年最低提款。 |
| HBP | 首次购房者计划 | Home Buyers' Plan，符合条件时可从 RRSP 提款用于买房，并按规则还款。 |
| LLP | 终身学习计划 | Lifelong Learning Plan，符合条件时可从 RRSP 提款用于本人或配偶教育培训，并按规则还款。 |

## 投资

| Term | 中文 | 说明 |
| --- | --- | --- |
| Asset allocation | 资产配置 | 股票、债券、现金和其他资产的比例安排。 |
| Diversification | 分散投资 | 降低单一公司、行业、地区、货币或资产类别风险。 |
| Rebalancing | 再平衡 | 把投资组合比例调回目标配置。 |
| GIC | 保本存款证 | Guaranteed Investment Certificate，通常有固定期限和利率。 |
| ETF | 交易所交易基金 | 在交易所买卖的基金，可追踪指数或策略。 |
| Mutual fund | 共同基金 | 集合投资产品，通常由基金经理管理。 |
| MER | 管理费用率 | Management expense ratio，基金每年的总管理费用比例。 |
| Capital gain | 资本利得 | 资本资产卖出价格高于成本时产生的收益。 |
| ACB | 调整后成本基础 | Adjusted cost base，用于计算资本利得或亏损。 |

## 保险

| Term | 中文 | 说明 |
| --- | --- | --- |
| Term life | 定期寿险 | 固定期限的人寿保障。 |
| Whole life | 终身寿险 | 永久寿险，通常有现金价值。 |
| Universal life | 万能寿险 | 结合人寿保障和投资/现金价值组件的永久寿险。 |
| Critical illness | 重疾险 | 符合合同定义的重大疾病发生后给付。 |
| Disability insurance | 伤残险 | 因疾病或意外无法工作时替代部分收入。 |
| Annuity | 年金 | 用本金换取定期收入，常用于退休现金流。 |
| Underwriting | 核保 | 保险公司评估健康、职业、生活方式和风险。 |
| Beneficiary | 受益人 | 保单或账户指定接收利益的人。 |
| Exclusion | 不保事项 | 合同列明不会赔付的情况。 |

## 房贷

| Term | 中文 | 说明 |
| --- | --- | --- |
| Mortgage | 房贷/按揭 | 以房产作为抵押的贷款。 |
| Down payment | 首付 | 买房时借款人先支付的金额。 |
| Term | 房贷合同期 | 当前房贷合同有效期间。 |
| Amortization | 摊还期 | 按当前还款安排预计还清贷款的总年限。 |
| Fixed rate | 固定利率 | term 内利率不变。 |
| Variable rate | 浮动利率 | 利率会随基准变化。 |
| Renewal | 续约 | term 到期后重新签新 term。 |
| Refinance | 再融资 | 重新安排贷款，可能增加贷款、取 equity 或改变 amortization。 |
| Prepayment privilege | 提前还款权利 | 不触发罚金时可额外还本金的额度或规则。 |
| Stress test | 压力测试 | 用高于合同利率的 qualifying rate 测试借款人承受能力。 |

## 税务和企业

| Term | 中文 | 说明 |
| --- | --- | --- |
| Deduction | 扣除 | 减少应税收入的项目。 |
| Credit | 税务抵免 | 减少税额的项目。 |
| Marginal tax rate | 边际税率 | 下一加元收入适用的大致税率。 |
| GST/HST | 商品及服务税/统一销售税 | 企业可能需要注册、收取和申报的销售税。 |
| Payroll remittance | 工资代扣代缴 | 雇主向 CRA 缴纳所得税、CPP、EI 等扣款。 |
| Salary | 工资 | 公司支付给 owner 或员工的就业收入。 |
| Dividend | 股息 | 公司向股东分配税后利润。 |
| Shareholder agreement | 股东协议 | 约定股东权利、退出、死亡、伤残、买卖安排的法律文件。 |

## 传承

| Term | 中文 | 说明 |
| --- | --- | --- |
| Will | 遗嘱 | 死亡后资产分配和 executor 安排的法律文件。 |
| Executor | 遗嘱执行人 | 执行遗嘱、处理债务税款并分配遗产的人。 |
| Probate | 遗嘱认证 | 法院确认遗嘱有效和 executor 权限的程序。 |
| Enduring Power of Attorney | 持久授权书 | 指定人在本人失能时处理财务和法律事务。 |
| Representation Agreement | 代表协议 | 在 BC 用于指定健康、个人护理等决定代表的文件。 |
| Advance Directive | 预先医疗指示 | 对未来医疗处理表达预先决定的文件。 |
