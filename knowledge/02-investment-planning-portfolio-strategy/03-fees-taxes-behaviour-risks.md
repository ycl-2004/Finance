# 费用、税务和行为风险

## 税务位置

同一个投资放在不同账户，税后结果可能不同。

- TFSA：合规收益和提款一般免税，供款不抵税。
- RRSP：供款可能抵税，提款通常作为收入纳税。
- FHSA：合格供款可抵税，合格购房提款免税。
- RESP：供款不抵税，教育援助付款通常由学生纳税。
- RDSP：供款不抵税，政府补助和投资收益提款时可能计入受益人收入。
- 非注册账户：利息、股息、资本利得按各自税务规则处理。

## 投资收入类型

- 利息：通常按普通收入纳税。
- 股息：加拿大 eligible dividend 和 non-eligible dividend 有不同税务处理。
- 资本利得：卖出资产高于成本时产生，只有应税部分计入收入。
- 外国收入：可能有预扣税和报税要求。
- Return of capital：可能降低 ACB，未来影响资本利得计算。

## 行为风险

常见行为错误：

- 市场上涨后觉得风险消失。
- 市场下跌后把长期计划变成短期恐慌。
- 看到朋友赚钱就追高。
- 只看单个产品收益，不看整体家庭资产。
- 把投资当成短期赌博。
- 因为账户免税就忽略投资风险。

## 投资前检查

- 我知道这个产品是什么法律形式吗。
- 我知道它主要投资什么吗。
- 我知道最坏情况下会亏多少吗。
- 我知道什么时候可以取钱、有无罚金或限制吗。
- 我知道所有费用吗。
- 我知道收益怎么纳税吗。
- 我知道这个投资服务哪个目标吗。

## 顾问适合性资料

持牌顾问做投资建议前通常需要了解：

- 身份和基本背景。
- 投资目标。
- 投资知识和经验。
- 风险承受能力。
- 时间周期。
- 财务状况。
- 流动性需求。
- 税务情况。

如果没有这些资料就推荐产品，风险很高。

## 客户怎么取得投资适合性资料

客户需要把“我想赚钱”拆成可核对资料。

| 要核对的点 | 客户去哪里拿 | 要拿什么 |
| --- | --- | --- |
| 投资账户 | 银行、dealer、投资平台、advisor portal | 最新 account statement、上一年年底 statement |
| 持仓 | 投资平台 holdings 页面 | Holdings report、资产类别、币种、集中度 |
| 成本和税务 | 投资平台、tax package、会计师 | Book cost/ACB、T3、T5、T5008、realized gain/loss report |
| 交易历史 | 投资平台 transaction history | 买入、卖出、分红再投资、转入转出 |
| 费用 | Dealer/advisor、annual report | Fee report、management fee、MER、trading commission |
| 目标和时间 | 客户本人 | 每个目标的金额、使用时间、优先级 |
| 风险承受能力 | 顾问 KYC/risk questionnaire | 风险问卷、投资经验、最大可承受下跌 |
| 流动性需求 | 银行 statement、目标清单 | 未来 0 到 5 年要用的钱 |

操作步骤：

1. 列出所有投资账户，不管金额大小。
2. 下载每个账户最新 statement 和上一年年底 statement。
3. 下载 holdings report，确认每个产品名称、代码、比例和账户类型。
4. 非注册账户另下载 realized gain/loss、T5008、T3/T5 和 ACB 资料。
5. 下载 fee report 或 annual charges and compensation report。
6. 把每笔钱对应到目标：应急金、买房、退休、教育、企业现金、长期投资。
7. 对不懂的产品，先确认法律形式和流动性，再讨论风险。

注意：

- 客户自己口头说“保守”或“激进”不够，要结合收入稳定性、负债、时间、目标和心理承受能力。
- 非注册账户的税务资料要和会计师核对，尤其是 ACB、转仓和 return of capital。
- 企业账户和个人账户要分开看，不能把公司运营现金当个人长期投资资金。

全局资料取得流程见 `../client-document-source-map.md`。
