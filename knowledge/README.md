# QM Financials 中文金融知识库

这个目录现在以业务主题为主入口。每个主题都有独立资料夹，资料夹内用小型 Markdown 文件保存可检索的金融知识。

内容目标：

- 面向小白学习，不假设读者已有加拿大金融经验。
- 只写概念、规则、流程、风险点和需要向专业人士确认的问题。
- 不写产品推荐、不写具体投资建议、不承诺税务或贷款结果。
- 正文尽量不插外部链接；使用到的来源统一放在每个 Markdown 文件底部，方便学习时先读干货、最后再核对来源。

资料校准时间：2026-04。会变动的限额、税率、贷款规则、政府补助和监管要求，实际使用前要重新核对。
